# 🎄 Advent of Code 🌟

## 🦌 Running the code

```
bun test --watch 2020/1
```

Applies to `2020` and `2023` onwards

### 🤶🏻 Old execution

```
bun ./2022/6/a.ts --test
bun ./2022/6/a.ts -t
bun ./2022/6/a.ts
```

You **must** place `input.txt` and `testInput.txt` in the same directory as your code

### 🎅🏻 Oldest excution

For JS files:

```
node ./2022/1/a.js
```

For TS files:

```
bun ./2022/5/a.ts
```
